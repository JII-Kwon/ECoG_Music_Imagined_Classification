# ECoG_Music_Imagined_Classification

## Overview

This archive contains the custom Python code used for preprocessing, feature extraction, decoding, and analysis in the study of "Reconstruction of Imagined Melody with Relative Pitch Decoding in Electrocorticography" [in preparation].

The code is provided as Extended Data for peer review in accordance with eNeuro policy on computational neuroscience models.
